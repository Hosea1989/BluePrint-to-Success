Introduction

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the foundation of every website. HTML provides structure, CSS provides style.

HTML Basics

  • Elements: Tags that define content - <h1>, <p>, <div>, <span>
  • Attributes: Additional info - class, id, href, src
  • Semantic HTML: Meaningful tags - <header>, <nav>, <main>, <article>
  • Forms: User input - <input>, <select>, <textarea>

CSS Fundamentals

  • Selectors: Target elements - element, class, ID, attribute
  • Properties: What to style - color, font-size, margin, padding
  • Box Model: Content, padding, border, margin
  • Specificity: Which styles take precedence

Layout Techniques

  • Flexbox: One-dimensional layouts (rows or columns)
  • CSS Grid: Two-dimensional layouts (rows AND columns)
  • Positioning: static, relative, absolute, fixed, sticky

Responsive Design

  • Media Queries: Apply styles based on screen size
  • Mobile-First: Design for mobile, then scale up
  • Fluid Typography: Text that scales with viewport
  • Flexible Images: max-width: 100% for responsive images

Modern CSS

  • CSS Variables (Custom Properties)
  • CSS Animations and Transitions
  • CSS Frameworks (Tailwind, Bootstrap)
