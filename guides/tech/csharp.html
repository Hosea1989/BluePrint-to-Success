Introduction

C# (C-Sharp) is a modern, object-oriented language developed by Microsoft. It's the primary language for .NET development and game development with Unity.

Why Learn C#?

  • Unity game development (most popular game engine)
  • Windows desktop applications
  • Web development with ASP.NET
  • Cross-platform with .NET MAUI

Core Concepts

  • Types: Value types vs reference types
  • Classes: Object-oriented programming foundation
  • Properties: Getters and setters
  • LINQ: Query syntax for collections
  • Async/Await: Asynchronous programming

.NET Ecosystem

  • ASP.NET: Web applications and APIs
  • Unity: 2D/3D game development
  • Xamarin/.NET MAUI: Cross-platform mobile apps
  • Entity Framework: Database ORM

Learning Path

  1. Weeks 1-2: C# fundamentals and syntax
  2. Weeks 3-4: Object-oriented programming
  3. Weeks 5-8: Choose specialization (Unity, Web, Desktop)
