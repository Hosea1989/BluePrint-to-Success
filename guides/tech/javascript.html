Introduction

JavaScript is the programming language of the web. It powers interactive websites, web apps, mobile apps, and even server-side applications.

Core Concepts

  • Variables: let, const, var - storing data
  • Data Types: strings, numbers, booleans, arrays, objects
  • Functions: Reusable blocks of code
  • Control Flow: if/else, switch, loops (for, while)

ES6+ Features

  • Arrow Functions: Concise function syntax
  • Template Literals: String interpolation with backticks
  • Destructuring: Extract values from objects/arrays
  • Spread/Rest: ... operator for arrays and objects
  • Promises/Async-Await: Handle asynchronous operations

DOM Manipulation

  • Selecting Elements: querySelector, getElementById
  • Modifying Content: textContent, innerHTML
  • Event Listeners: addEventListener for interactivity
  • Creating Elements: createElement, appendChild

JavaScript Frameworks

  • React: Component-based UI library (most popular)
  • Vue: Progressive framework (beginner-friendly)
  • Angular: Full framework (enterprise focus)
  • Node.js: JavaScript runtime for backend

  4. Quiz game
